# Рассылки в Viber

Viber-рассылки — это выпуски, отправленные от лица компании тем пользователям, кто установил приложение Viber и дал согласие на получение рассылок. Отправка Viber-рассылок подключается и оплачивается отдельно на платных аккаунтах юридических лиц: чтобы активировать эту дополнительную возможность на вашем аккаунте, <SupportLink>напишите в чат поддержки</SupportLink>.

Для работы с Viber-рассылками необходимо заключить дополнительное соглашение и зарегистрировать официальное имя отправителя. Также можно [подключить SMS-канал](https://docs.sendsay.ru/other-channels/sms/how-to-connect-sms/), чтобы создать мультиканальный формат коммуникации с пользователями: одновременная работа с Viber и SMS позволяет отправлять рассылки в тот канал связи, который удобен клиентам.

Контакты пользователей Viber хранятся в системе под видом номера телефона. Перед отправкой сообщений система проверяет контакт: если номер пользователя привязан к Viber, он получит рассылку в Viber. Если же номер телефона не привязан к Viber, сообщение не отправится, либо отправится как смс-рассылка (если в аккаунте подключен SMS-канал).

## Как подключить бизнес-рассылки в Viber

При подключении бизнес-рассылок в Viber необходимо заполнить специальную анкету регистрации. Для этого понадобится следующая информация:

- название компании на языке оригинала и на английском языке,
- адрес сайта компании,
- телефон компании для профиля (должен совпадать с контактами на сайте),
- страну регистрации и местонахождения,
- в какие страны будут отправляться сообщения,
- тип предоставляемых услуг (продажи, страхование, финансы и т. д.),
- прогноз объёма трафика (количество сообщений в день),
- ожидаемый пик нагрузки (сообщений в секунду),
- характер трафика (личные данные, публичные данные, реклама и т. д.),
- ссылка или скриншот на оферту/условия оказания услуг с информацией о получении абонентом сообщений по Viber,
- текстовое описание компании (10-400 символов без ссылок),
- примеры текста для рекламной, транзакционной и приветственной рассылки.

:::tip Важно
Для подключения рассылок в Viber необходимо подтвердить гарантии юридического лица в отношении отправки верифицированных сообщений. Если юридическое лицо нарушит гарантии, ему придётся заплатить штраф.
:::

## Как отправить рассылку в Viber

Отправить рассылку в Viber можно двумя способами:

1. В интерфейсе, если сообщение содержит только текст.
2. Через [Sendsay API](https://sendsay.ru/api/api.html), если сообщение помимо текста содержит картинку и/или кнопку.

Для отправки Viber-рассылки в интерфейсе Sendsay воспользуйтесь [механикой отправки смс-сообщения](https://docs.sendsay.ru/other-channels/sms/how-to-create-sms-campaign): перейдите в раздел [**Рассылки**](https://app.sendsay.ru/campaigns/issues), нажмите «Создать выпуск» и выберите тип SMS.

Если же вы хотите добавить к сообщению картинку и/или кнопку, перейдите в [API-консоль](https://app.sendsay.ru/api/console) и укажите нужные параметры вызова `issue.send`. Например:

```
{
	"action": "issue.send",
	"group": "id списка",
	"letter": {
		"button.url": "ссылка на кнопке",
		"button.text": "текст на кнопке",
		"message": {
			"viber": "текст cообщения"
		}
	},
	"attaches": [
		{
			"url": "ссылка на картинку http/https/ftp/ftps/sftp"
		}
	]
}
```

[Отправка выпуска в Sendsay API](https://sendsay.ru/api/api.html#Отправить-выпуск)
