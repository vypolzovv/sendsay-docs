# Прогрев IP-адресов

**Прогрев IP-адреса** — это постепенное увеличение количества отправленных email-рассылок с нового IP-адреса. Задача этого процесса&nbsp;&mdash; познакомить почтовых провайдеров с&nbsp;привычными объёмами отправлений. Так формируется позитивная репутация отправителя, которая влияет на&nbsp;доставляемость писем.

Существуют понятия прогрева как IP-адреса, так и&nbsp;домена&nbsp;&mdash; технически это один и&nbsp;тот&nbsp;же процесс. Прогрев необходим, если вы&nbsp;используете:

- старый домен и&nbsp;новый IP-адрес,
- старый IP-адрес и&nbsp;новый домен,
- новый IP-адрес и&nbsp;новый домен.

:::tip Важно
Если у&nbsp;вас ещё нет базы контактов и&nbsp;ваш email-маркетинг только на&nbsp;старте, органический прирост подписчиков создаст идеальные условия для IP-прогрева. Новые контакты в&nbsp;базе будут позитивно влиять на&nbsp;репутацию отправителя без какого-либо вмешательства.
:::

Основное правило прогрева&nbsp;&mdash; это плавность. Если начать отправлять большое количество писем с&nbsp;непрогретого домена или IP-адреса, почтовые провайдеры могут перестать принимать ваши рассылки: они будут считать вас ненадёжным отправителем, ведь истории отправок ещё нет. Неправильная работа с&nbsp;прогревом может привести к&nbsp;тому, что письма попадут в&nbsp;спам, а&nbsp;сам домен попадет под блокировку.

Отправка рассылок в&nbsp;Sendsay возможна с&nbsp;IP-адресов, которые различаются по&nbsp;степени приватности:

- **общий IP-адрес**&nbsp;&mdash; с&nbsp;него отправляют письма несколько клиентов Sendsay, и&nbsp;репутация одного отправителя зависит от&nbsp;остальных. Мы&nbsp;внимательно следим за&nbsp;тем, что отправляем, поэтому почтовые провайдеры нам доверяют и&nbsp;не&nbsp;блокируют рассылки. Однако если вы&nbsp;используете публичный IP-адрес в&nbsp;связке с&nbsp;новым доменом, прогрев всё равно необходим;
- **выделенный IP-адрес** привязывается к&nbsp;одному аккаунту Sendsay, а&nbsp;за&nbsp;его репутацию отвечает единственный отправитель. В&nbsp;начале работы с&nbsp;выделенным IP-адресом необходимо выполнить его прогрев.

:::tip Важно
За&nbsp;дополнительную плату предоставляем от&nbsp;двух выделенных IP-адресов&nbsp;&mdash; так вы&nbsp;будете уверены, что репутация чистая и&nbsp;зависит только от&nbsp;вас. Если хотите подключить выделенный IP-адрес, напишите в&nbsp;техподдержку.
:::

## Пошаговая инструкция: как прогреть IP-адрес

### 1. Настройте email-аутентификацию

Пропишите [ключ DKIM, SPF-записи и&nbspDMARC-политику](https://docs.sendsay.ru/email-campaigns/settings/email-authentication/). Вы&nbsp;можете сделать это как самостоятельно, так и&nbsp;заказать полную настройку у&nbsp;специалистов Sendsay.

[Как настроить email-аутентификацию](https://docs.sendsay.ru/email-campaigns/settings/how-to-connect-domain/#способы-настройки-email-аутентификации)

### 2. Актуализируйте базу контактов

При прогреве важно наращивать количество отправленных рассылок постепенно и&nbsp;не&nbsp;отступать от&nbsp;плана. Поэтому нужно отправлять рассылки &laquo;пачками&raquo; вовлечённым подписчикам&nbsp;&mdash; тем, кто с&nbsp;наименьшей вероятностью отправит рассылку в&nbsp;спам.

Чтобы актуализировать базу, исключите пассивных и&nbsp;&laquo;спящих&raquo; пользователей, которые не&nbsp;открывали ваши рассылки на&nbsp;протяжении трёх&mdash;четырёх месяцев. В&nbsp;результате база может сократиться, зато в&nbsp;ней будут только активные контакты, заинтересованные в&nbsp;вашей рассылке. Всё это повысит уровень открытий и&nbsp;снизит количество жалоб, что важно для прогрева и&nbsp;положительной репутации отправителя у&nbsp;почтовых провайдеров.

:::tip Важно
Общее количество контактов для прогрева должно равняться количеству писем, которые вы&nbsp;планируете отправить.
:::

### 3. Подготовьте нейтральное письмо

Для прогрева IP-адреса лучше использовать нейтральную рассылку&nbsp;&mdash; остановитесь на&nbsp;полезном контенте, который вызывает хороший отклик у&nbsp;читателей (например, дайджест). Важно:

- наличие рабочей ссылки отписки,
- рекомендуемый вес HTML-кода письма — до 102 Кб, чтобы оно не обрезалось в Gmail,
- в рассылке должна быть корректная и чистая вёрстка, которая хорошо отображается на разных устройствах.

Подготовьте черновик письма для прогрева: в боковом меню перейдите в раздел **Рассылки → Черновики** и нажмите синюю кнопку «+ Создать выпуск». Далее на шаге Письмо создайте или загрузите шаблон рассылки.

[Как создать письмо в блочном редакторе](https://docs.sendsay.ru/email-campaigns/create-your-campaign/drag-and-drop-editor)

[Как загрузить HTML-шаблон](https://docs.sendsay.ru/email-campaigns/create-your-campaign/how-to-upload-html-template)

### 4. Составьте график отправки писем

Теперь нужно определиться с&nbsp;количеством отправляемых рассылок. Главное&nbsp;&mdash; не&nbsp;торопиться: отправляйте небольшие объёмы писем самым активным подписчикам, постепенно увеличивая количество получателей.

:::tip Важно
Универсального графика увеличения объёмов рассылок нет: оптимальная скорость зависит от&nbsp;того, насколько давно вы&nbsp;собираете подписную базу, активности получателей и&nbsp;ряда других факторов.
:::

Чтобы прогрев был плавным, ежедневно увеличивайте объём рассылок не&nbsp;более чем на&nbsp;50% к&nbsp;предыдущему дню. Например, вы&nbsp;начинаете с&nbsp;1&nbsp;000 писем в&nbsp;день&nbsp;&mdash; тогда формула для второго дня отправки будет выглядеть так:

```
1 000 + (1 000 * 0,5) = 1 500
```

А&nbsp;для третьего дня&nbsp;&mdash; так:

```
1 500 + (1 500 * 0,5) = 2 250
```

### 5. Настройте отправку писем

При прогреве используйте встроенную опцию отправки выпуска по&nbsp;частям. Для этого:

1. Создайте выпуск: в&nbsp;боковом меню перейдите в&nbsp;раздел **Рассылки &rarr; Черновики** и&nbsp;выберите заранее подготовленный черновик нейтрального письма для прогрева.
2. На&nbsp;шаге Аудитория нажмите кнопку &laquo;Изменить&raquo; и&nbsp;выберите получателей рассылки.
3. На&nbsp;шаге Письмо нажмите кнопку &laquo;Изменить&raquo; и&nbsp;заполните поля &laquo;Имя отправителя&raquo;, &laquo;Email отправителя&raquo; и&nbsp;&laquo;Тема письма&raquo;. После этого кликните по&nbsp;синей кнопке &laquo;Сохранить&raquo;.
4. Откройте выпадающий список у&nbsp;кнопки &laquo;Отправить&raquo; и&nbsp;выберите пункт **Отправить по&nbsp;частям**.
5. Настройте параметры отправки: пропишите количество писем для первого дня отправки, установите периодичность&nbsp;&mdash; один раз в&nbsp;сутки, а&nbsp;также поставьте галочку в&nbsp;пункте &laquo;Прогрессивно увеличивать размеры частей&raquo;, установив&nbsp;50% в&nbsp;раскрывающемся списке.
6. Когда параметры отправки выпуска по&nbsp;частям заданы, нажмите синюю кнопку &laquo;Выпустить&raquo;.<br />

### 6. Отслеживайте показатели

Регулярно отслеживайте статистику, чтобы определить оптимальную частоту рассылки. Какие показатели здесь важны:

- ошибки доставки (Ошибки&nbsp;%) должны быть в&nbsp;пределах 1&ndash;1,5%;
- соотношение отписок к&nbsp;открытиям (UTOR&nbsp;%) должно быть не&nbsp;больше 4%.

В&nbsp;Sendsay данные по&nbsp;этим метрикам можно посмотреть в&nbsp;разделе **Статистика**. Выберите нужный выпуск и&nbsp;в&nbsp;открывшейся статистике перейдите во&nbsp;вкладку **Доставка &rarr; Ошибки доставки**. Во&nbsp;вкладке **Отписки** можно посмотреть количество и&nbsp;процент пользователей, отписавшихся от&nbsp;рассылки через жалобу на&nbsp;спам.
