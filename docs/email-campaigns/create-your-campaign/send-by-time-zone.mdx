# Как отправить письмо по часовым поясам

Если ваши подписчики живут в разных регионах страны, следует понимать, что один контакт может получить письмо в&nbsp;10&nbsp;утра, а&nbsp;другой&nbsp;&mdash; в&nbsp;18&nbsp;вечера. В&nbsp;результате один прочитает письмо, а&nbsp;второй, возможно, и&nbsp;не&nbsp;откроет его.

В&nbsp;таком случае **отправка по&nbsp;часовым поясам** поможет добиться более высокой открываемости, ведь письма во&nbsp;Владивостоке прилетят к&nbsp;своим читателям утром, а&nbsp;не&nbsp;глубокой ночью, когда все спят (и&nbsp;вряд&nbsp;ли читают почту). Выпуск будет отправляться частями в&nbsp;указанное время&nbsp;&mdash; для каждого подписчика в&nbsp;его часовом поясе.

При отправке писем по&nbsp;часовым поясам система определяет время отправки в&nbsp;зависимости от&nbsp;города, указанного в&nbsp;базовой анкете подписчика в&nbsp;поле `Город`.

:::tip Важно
Если поле `Город` в&nbsp;анкете подписчика не&nbsp;заполнено или система не&nbsp;смогла определить часовой пояс, выпуск отправится в&nbsp;указанное время по&nbsp;Москве (UTC+3).
:::

Чтобы отправить рассылку по&nbsp;часовым поясам:

1. В&nbsp;боковом меню перейдите в&nbsp;раздел **Рассылки** и&nbsp;создайте выпуск. Заполните шаги **Аудитория**, **Письмо** и&nbsp;**Дополнительные настройки** (если нужно).

   [Как отправить массовую email-рассылку](https://docs.sendsay.ru/email-campaigns/create-your-campaign/how-to-send-email-campaign)

2. На шаге **Отправка** выберите тип **Отправить по часовым поясам**, затем нажмите «Далее»:

   ![Send by time zone](/img/email-campaigns/create-your-campaign/send-by-time-zone/send-by-time-zone.png)

3. Настройте время отправки: выберите время, в&nbsp;которое подписчики получат письма в&nbsp;своих часовых поясах. Здесь&nbsp;же можно посмотреть прогноз времени доставки в&nbsp;разных городах:

   ![How to send by time zone](/img/email-campaigns/create-your-campaign/send-by-time-zone/how-to-send-by-time-zone.gif)

   :::tip Важно
   Система работает по&nbsp;московскому времени&nbsp;&mdash; и&nbsp;если в&nbsp;Москве 12&nbsp;дня, то&nbsp;во&nbsp;Владивостоке уже 19&nbsp;часов вечера. Если настроить время отправки на&nbsp;15&nbsp;часов, подписчики во&nbsp;Владивостоке получат письма в&nbsp;15&nbsp;часов следующего дня.

   Чтобы выпуски отправились по&nbsp;разным часовым поясам в&nbsp;один день, лучше планировать отправку заранее&nbsp;&mdash; как минимум за&nbsp;24&nbsp;часа.
   :::

4. Когда время отправки задано, нажмите &laquo;Запланировать&raquo;. После этого письмо попадёт в&nbsp;папку **Рассылки &rarr; Запланированные**.

Есть и&nbsp;другой способ отправки рассылки с&nbsp;более точным определением часового пояса. Для этого в&nbsp;анкете подписчика необходимо в&nbsp;любом поле указать числовые смещения по&nbsp;времени относительно UTC в&nbsp;формате `UTC+h` или `UTC-h`. Например, для города Мирный в&nbsp;Якутии запись может выглядеть так:

<p align="center">
  <img width="50%" src={fillTheForm} alt="Start event" />
</p>

Чтобы отправить рассылку с&nbsp;более точным определением часового пояса:

1. Перейдите [в&nbsp;предыдущий интерфейс](https://sendsay.ru/account/).
2. В&nbsp;разделе **Рассылки** создайте выпуск и&nbsp;нажмите &laquo;Перейти к&nbsp;отправке&raquo;.
3. На&nbsp;шаге **Когда отправить выпуск** выберите пункт **Отправлять письма по&nbsp;временной зоне подписчика**.
4. Установите час, в&nbsp;который выйдет рассылка.
5. На&nbsp;шаге **Источник** укажите поле, по&nbsp;которому система определит часовой пояс.

<p align="center">
  <img width="50%" src={specifyField} alt="Start event" />
</p>

6. Если часовой пояс указан в&nbsp;разных полях анкеты подписчиков, нажмите &laquo;+ добавить источник&raquo;
   и&nbsp;выберите ещё одно или несколько полей, откуда система сможет подтянуть указанные данные о&nbsp;часовом
   поясе. Затем нажмите &laquo;Готово&raquo;.

## Как изменить время отправки письма по&nbsp;часовым поясам

1. Перейдите в&nbsp;раздел **Рассылки &rarr; Запланированные** и&nbsp;выберите нужную рассылку.
2. Нажмите &laquo;Редактировать&raquo; и&nbsp;выберите пункт **Изменить время**.
3. Выберите день и&nbsp;время, когда подписчики получат письма в&nbsp;своих часовых поясах.

   ![How to change the send time](/img/email-campaigns/create-your-campaign/send-by-time-zone/how-to-change-send-time.gif)

4. Нажмите &laquo;Изменить&raquo;.

Чтобы приостановить отправку рассылок по&nbsp;часовым поясам, нажмите &laquo;Редактировать&raquo; и&nbsp;выберите пункт **Приостановить**. В&nbsp;случае если нужно снова запланировать отправку этой рассылки, перейдите в&nbsp;настройки письма и&nbsp;нажмите &laquo;Запланировать&raquo;.

**Читайте также:** [Как запустить оптимизацию времени рассылки](https://docs.sendsay.ru/email-campaigns/create-your-campaign/send-time-optimization)
