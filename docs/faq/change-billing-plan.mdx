# Как перейти на другой тариф

В Sendsay есть четыре линейки тарифов: **Рассылки**, **Маркетинг**, **Транспорт** и **Хранилище (CDP)**&nbsp;&mdash; все они описаны [на&nbsp;странице Тарифы](https://sendsay.ru/rates).

Если вы&nbsp;используете тариф, возможности которого оказались избыточными для решения ваших маркетинговых задач, вы&nbsp;можете перейти на&nbsp;любой другой тариф с&nbsp;меньшим функционалом. И&nbsp;наоборот: если допустимых лимитов на&nbsp;текущем тарифе не&nbsp;хватает&nbsp;&mdash; их&nbsp;можно увеличить, перейдя на&nbsp;тариф с&nbsp;б&oacute;льшими возможностями.

:::tip Важно
В&nbsp;личном кабинете Sendsay вы&nbsp;можете самостоятельно сменить тариф внутри линеек **Маркетинг** и&nbsp;**Рассылки**, а&nbsp;также перейти с&nbsp;тарифа **Маркетинг** на&nbsp;**Рассылки** и&nbsp;наоборот. Если вы&nbsp;хотите сменить другой тариф, <SupportLink>напишите в чат поддержки</SupportLink>.
:::

Чтобы самостоятельно сменить тариф, откройте меню аккаунта и&nbsp;выберите пункт **Оплата &rarr; Тарифы**. У&nbsp;каждого тарифа есть свои возможности и&nbsp;ограничения: система распознает, если вы&nbsp;захотите перейти на&nbsp;тариф с&nbsp;б&oacute;льшими лимитами или наоборот&nbsp;&mdash; с&nbsp;меньшим функционалом.

## Если есть превышения по&nbsp;количеству контактов

Например, ваш тариф предусматривает до&nbsp;50&nbsp;000 подписчиков в&nbsp;базе контактов, но&nbsp;вы&nbsp;хотите перейти на&nbsp;тариф с&nbsp;меньшей базой:

<p align="center">
  <img width="70%" src={excessContacts} alt="Excess contacts" />
</p>

**Удалите из&nbsp;базы контакты**, число которых превышает лимит нового тарифа. Либо вы&nbsp;можете выбрать другой тариф&nbsp;&mdash; с&nbsp;б&oacute;льшей базой.

## Если есть превышения по&nbsp;лимитам

Например, вы&nbsp;хотите перейти с&nbsp;тарифа **Маркетинг М50**&nbsp;на тариф **Рассылки Р5**&nbsp;&mdash; тогда система подскажет об&nbsp;ограничениях:

<p align="center">
  <img width="70%" src={excessLimits} alt="Excess limits" />
</p>

Для перехода на&nbsp;новый тариф необходимо исключить используемые функции, число которых превышает лимиты нового тарифа:

- **удалить** контакты из&nbsp;базы,
- **деактивировать** триггерные автоматизации и&nbsp;автоматизации по&nbsp;времени,
- **удалить** пользователей аккаунта, домены и&nbsp;адреса отправителей.

:::tip Важно
Если при переходе на&nbsp;другой тариф нет превышений по&nbsp;лимитам, текущие автоматизации продолжат работать. Активировать новые можно **в&nbsp;пределах лимитов**.
:::

## Если есть остаток от&nbsp;текущего тарифа в&nbsp;денежном эквиваленте

Например, вы&nbsp;приобрели тариф на&nbsp;месяц, не&nbsp;израсходовали его лимиты,&nbsp;&mdash; и&nbsp;теперь хотите сменить тариф. В&nbsp;таком случае система подсчитает остаток от&nbsp;текущего тарифа в&nbsp;денежном эквиваленте и&nbsp;зачтёт ранее оплаченное в&nbsp;итоговую сумму:

<p align="center">
  <img width="40%" src={remainingBalance} alt="Remaining balance" />
</p>

Таким образом за&nbsp;переход на&nbsp;новый тариф необходимо заплатить меньшую сумму&nbsp;&mdash; с&nbsp;вычетом остатка от&nbsp;итоговой суммы.

## Если остаток от&nbsp;текущего тарифа в&nbsp;денежном эквиваленте слишком большой

Такая ситуация может возникнуть, например, если вы&nbsp;уже оплатили годовой тариф, но&nbsp;хотите перейти на&nbsp;меньший тариф&nbsp;&mdash; и&nbsp;в&nbsp;пересчёте на&nbsp;деньги на&nbsp;вашем счету получится большой остаток.

В&nbsp;таком случае самостоятельно перейти на&nbsp;другой тариф не&nbsp;получится: чтобы сменить тариф, <SupportLink>напишите в чат поддержки</SupportLink>.
